Simply link to paper.min.css (the bootstrap file included here is just for this demo page, and an .scss file is also included for those wishing more advanced customization).

The following classes are available in addition to the default .paper class. Include the default .paper class along with optional paper treatment and color. So for example you might have:

<div class="paper paper-curl paper-vintage"> <div class="paper paper-polar"> <div class="paper paper-curve-vert">

Add optional .paper-sm or .paper-lg classes for sizing. Note that these paper pieces have the same basic styling (padding, margin, and min-height) as Bootstrap wells. They expand to fit whatever they're put inside (for example, Bootstrap grid columns), and can accomodate any type and amount of content.

Depending on your content, you may wish to add .paper > p:last-child { margin-bottom: 0; } to your stylesheet (I do this for Bootstrap's wells too; it removes the last paragraph's 10px margin-bottom).

These are supported on all modern desktop and mobile browsers. IE8 supports none of these effects, but still gets basic styling (padding, margin), along with color and sizing options.
